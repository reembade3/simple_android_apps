# simple_android_apps
A collection of simple Android applications built with Kotlin, including Quiz App, BMI Calculator, and Stopwatch, focusing on basic UI, user input handling, and core Android concepts.
# Simple Android Apps

## Projects

### Calculator App
- Performs basic arithmetic operations.
- Simple UI with user input handling.

### BMI Calculator
- Calculates Body Mass Index (BMI) based on height and weight.
- Displays BMI result clearly to the user.

### Stopwatch App
- Basic stopwatch functionality.
- Includes start, stop, and reset actions.

## Technologies Used
- Kotlin  
- XML Layouts  
- Android Studio  
- Git & GitHub  

## Purpose
These projects were created to strengthen Android fundamentals such as:
- UI design and layouts  
- User input handling  
- Basic app logic  
- Activity lifecycle
